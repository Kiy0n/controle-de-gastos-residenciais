# Controle de Gastos Residenciais

Este projeto é uma API REST desenvolvida em ASP.NET Core como parte de um desafio técnico para uma vaga de estágio em Desenvolvimento C# .NET Full Stack.

O objetivo é praticar conceitos de desenvolvimento backend, arquitetura de APIs e boas práticas com C# e .NET.

---

## Objetivo do sistema

O sistema tem como finalidade controlar gastos residenciais, permitindo:

- Cadastro de pessoas
- Registro de transações financeiras
- Consulta de totais e saldos por pessoa

---

## Tecnologias utilizadas

- .NET 10
- ASP.NET Core Web API
- C#
- Entity Framework Core (a ser implementado)
- SQLite (a ser implementado)

---

## Como executar o projeto

```bash
dotnet restore
dotnet run
```

A API estará disponível em:

```
http://localhost:5044
```
